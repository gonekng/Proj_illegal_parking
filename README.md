##  시민수요를 반영한 불법주정차 단속 모델 개발

2021 공공빅데이터 청년인턴 교육기간 중 진행한  
**서울시 불법주정차 단속 우선지역 최적화 프로젝트**이며,

저는 **데이터 수집 및 전처리** 과정을 주로 맡아 진행하였습니다.

 **팀원:** 김상원(팀장), 강지원, 박이현, 이현준, 장한솔, 장효주, 최지혜  

 **사용 언어:** `R`  
 **스킬 및 작업 툴:** `Rstudio` `QGIS`
